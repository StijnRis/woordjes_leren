# Back end

## To start (type in powershell):

Set-ExecutionPolicy Unrestricted -Scope Process
.\/back_end/venv/Scripts/Activate.ps1
python back_end/manage.py runserver

## Open Shell:

python manage.py shell

# Front end

## Install front end:

npm install or npm i

## Start front end:

1. Open terminal
2. cd front_end
3. npm run dev
